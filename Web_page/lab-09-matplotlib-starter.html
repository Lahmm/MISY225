Matplotlib¶

There are 8 basic functions as described at http://mathonweb.com/help_ebook/html/functions_4.htm

Create a figure and do the following:

  1. use the seaborn style sheet
  2. create 4 subplots (2x2 2 rows with two subplots each row) with figure size (15, 15)
  3. plot the following 3 types of functions: Linear function, Power function, Quadratic function (you can create the specific function yourself) and
  • all functions can share the values of x, which should be 50 evenly spaced number between -10 and 10 (hint: use np.linspace())
  • add the function type as the title of the subplot
  • the quadratic function has to be in the subplot in the first row and second column (you leave one subplot empty and arrange the other two anyway you want
  • the functions must have different line style, marker style, and color
  • change the x label to x and y label to y
  • add legends for each subplot to show the functions

A sample output is as follows:
